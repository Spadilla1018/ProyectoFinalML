Machine Learning para Energía Sostenible (ODS 7)

El Machine Learning impulsa la ODS 7: Energía asequible y no contaminante mediante la predicción, optimización y análisis de datos energéticos, promoviendo el acceso universal a energía sostenible y eficiente.

Entendimiento del Negocio y los Datos

Regresión Lineal

La regresión lineal permite predecir el consumo energético en hogares o industrias a partir de variables como temperatura, ocupación o producción, facilitando la planificación y el ahorro de energía.

Regresión Logística

La regresión logística ayuda a clasificar instalaciones según su eficiencia energética o a predecir la probabilidad de adopción de tecnologías renovables, contribuyendo a la toma de decisiones sostenibles.

K-Nearest Neighbors (K-NN)

El algoritmo K-NN puede clasificar patrones de consumo energético, identificar usuarios con alto potencial de ahorro o segmentar zonas para implementar energías renovables de manera eficiente.
